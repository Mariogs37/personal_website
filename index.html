Squash Simulator: This application allows the user to simulate upcoming college squash matches. Simply select the two teams and the program will:

  • Scrape the latest team ladders and player ratings
  • Calculate approximate probability of each player winning based on the CSA's rating formula
  • Run as many simulations as the user would like
  • Generate a histogram to display the win / loss distribution graphically (see above)

Future Features:
